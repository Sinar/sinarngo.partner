sinarngo.partner Installation
-----------------------------

To install sinarngo.partner using zc.buildout and the plone.recipe.zope2instance
recipe to manage your project, you can do this:

* Add ``sinarngo.partner`` to the list of eggs to install, e.g.:

    [buildout]
    ...
    eggs =
        ...
        sinarngo.partner

* Re-run buildout, e.g. with:

    $ ./bin/buildout
